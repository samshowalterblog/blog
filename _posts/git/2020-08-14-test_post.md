######  安装git-remote-gcrypt和gnupg
```
sudo apt-get install git-remote-gcrypt gnupg
```
###### 创建一个gpg的key，
 需要设置用户名，邮箱，描述等，不要设置过期时间
```
gpg --gen-key
```
###### 记录一下生成的key的ID，

比如2048R/liberxue013里面的liberxue013，2048代表加密轮数，越多越不容易破解
```
gpg --list-keys
```
